# 工时管理系统简介 java927

## 项目概述

工时管理系统是一款轻量级的工时记录和管理工具，旨在帮助企业通过员工工时上报，准确记录项目所花费的工时，从而进行有效的项目工时统计和核算人工成本。系统提供实时、动态、真实的工时数据展示，助力企业项目管理。

## 技术栈

- 后端服务：Java 1.8 + Spring Boot
- 数据存储：MySQL
- 前端界面：Vue

## 系统架构

本项目基于前后端分离的架构设计，前端负责界面展示，后端提供数据处理服务。

## 功能模块

### 工时模块

- **我的工时**：提交工时、查看详情、修改工时、我的统计（填报率）。
- **项目统计**：项目日报、项目月报、项目日历、项目工时详情。
  - 项目成本
  - 项目投入进度
  - 项目工时总计
  - 工时明细
  - 项目详情
- **数据统计**：
  - 工时统计
  - 人员排行
  - 阶段统计
  - 工作内容统计
- **人员填报率统计**：人员填报率排名、填报详情。
- **工时审核**：工时审核设置、工时审批。
- **工时归档**：工时归档设置、工时归档。
- **消息提醒**：工时审核提醒。

### 管理模块

- **项目管理**：创建项目、添加人员、工时设置、状态管理、删除项目。
- **业务管理**：
  - 加班管理
  - 节假日管理
  - 工作类型管理
  - 审核管理
  - 归档管理
- **成本管理**：成本开关、岗位成本设置、人员成本设置。
- **组织管理**：用户管理、角色管理、重置密码、批量删除。
- **职位管理**：新增、修改、删除、批量删除等功能。

## 系统角色

- 员工：提交工时、查看工时详情与统计
- 审核员：负责工时的审核与审批
- 管理员：进行项目管理、业务管理、成本管理、组织管理及职位管理等

## 运行环境

- 服务器：兼容Java 1.8及以上版本的操作系统
- 数据库：MySQL
- 客户端：现代Web浏览器
